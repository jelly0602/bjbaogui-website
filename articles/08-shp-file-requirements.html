SHP文件生成要求与应用场景

介绍SHP文件的定义、要求和应用场景。

SHP服务应定位为空间矢量数据整理与核对，不替代具有资质的测绘单位出具法定测绘成果。SHP不是审批指定的唯一格式，而是按报审、测绘、核验和项目管理需要整理的空间矢量数据。

什么是SHP文件？

SHP文件（Shapefile）是一种矢量数据格式，由ESRI公司开发，广泛应用于GIS系统和规划管理。SHP文件包含地理空间数据，可用于地图展示、空间分析、规划管理等多种用途。

重要定位：SHP服务应定位为空间矢量数据整理与核对，其作用是将设计成果转化为规范化的空间数据格式，便于报审、测绘衔接和项目管理。SHP服务不替代具有资质的测绘单位出具法定测绘成果——法定测绘成果需由持证测绘单位按测绘规范出具。

SHP不是审批指定的唯一格式

需要澄清一个常见误解：SHP并非审批机关指定的唯一数据格式。实际上，SHP是按报审、测绘、核验和项目管理需要整理的空间矢量数据，在不同环节发挥不同作用：

  • 报审环节：配合BDB等审批系统要求的空间数据
  • 测绘衔接：与测绘单位的数据成果对接
  • 核验环节：用于竣工验收阶段的空间数据比对
  • 项目管理：为项目全周期的空间数据管理提供支撑

与多测合一的关系

北京推行"多测合一"改革，将建设工程从立项到竣工验收全流程的测绘事项整合。SHP数据在多测合一体系中的定位：

  • SHP整理工作可与多测合一的测绘成果衔接，确保空间数据的一致性
  • SHP数据可用于核对测绘成果与设计方案的偏差
  • 多测合一要求统一坐标系统和技术标准，SHP数据应遵循相同标准

文件组成

一个完整的SHP文件通常包含以下文件：

文件扩展名 说明
.shp 几何形状数据
.shx 几何索引文件
.dbf 属性数据表
.prj 坐标系统信息

数据内容

SHP文件应包含以下数据：

1. 用地红线

  • 项目用地范围
  • 红线拐点坐标
  • 红线尺寸标注

2. 建筑轮廓

  • 各建筑的轮廓线
  • 建筑名称、用途
  • 建筑面积、高度、层数

3. 道路系统

  • 道路中心线
  • 道路边线
  • 道路宽度、等级

4. 其他要素

  • 出入口
  • 停车设施
  • 绿化系统
  • 广场、水体、构筑物

6项质量检查

SHP文件在交付前应通过以下6项质量检查，确保数据准确可用：

检查项 检查内容
坐标检查 验证坐标系统是否正确，坐标值是否在合理范围内，是否存在偏移或异常值
闭合检查 验证多边形要素（用地红线、建筑轮廓等）是否闭合，首尾点是否一致
拓扑检查 验证要素间的空间关系是否正确，是否存在重叠、缝隙、自相交等拓扑错误
属性检查 验证属性表字段是否完整、数据类型是否正确、必填字段是否有值
版本检查 验证数据版本是否为最新版，是否与设计图纸一致，是否已更新审核意见
同源检查 验证SHP数据与PDF图纸、BDB文件是否同源，关键数据（面积、坐标、层数等）是否一致

要求项 具体要求
坐标系统 北京地方坐标系或2000国家大地坐标系
高程系统 北京地方高程系
文件格式 SHP格式
数据精度 符合规划管理要求

应用场景

1. 报审与测绘衔接

SHP文件在报审和测绘环节中用于：

  • 配合审批系统的空间数据要求
  • 与测绘单位的数据成果对接
  • 核对设计方案与测绘成果的一致性

2. 规划管理

SHP文件可以用于规划管理：

  • 用地管理
  • 建筑管理
  • 道路管理
  • 绿化管理

3. 竣工核验

SHP文件在竣工验收阶段用于：

  • 空间数据比对
  • 规划指标核验
  • 建设成果与审批方案的一致性检查

4. 项目全周期管理

SHP文件为项目全周期的空间数据管理提供支撑：

  • 立项阶段的现状数据
  • 设计阶段的方案数据
  • 施工阶段的进度数据
  • 竣工阶段的成果数据

常见问题

Q1: SHP文件是审批指定格式吗？

A: 不是。SHP不是审批机关指定的唯一数据格式。SHP是按报审、测绘、核验和项目管理需要整理的空间矢量数据，在不同环节发挥辅助作用。审批系统主要使用BDB格式。

Q2: SHP文件和BDB文件有什么区别？

A: BDB文件是北京市规划审批系统要求的专用格式文件，用于规划审批；SHP文件是空间矢量数据整理与核对的通用格式，服务于报审、测绘衔接和项目管理。

Q3: SHP文件可以用什么软件打开？

A: SHP文件可以用ArcGIS、QGIS、Global Mapper等GIS软件打开。

Q4: SHP文件需要什么坐标系统？

A: 通常采用北京地方坐标系或2000国家大地坐标系，具体要求根据使用目的确定。建议与测绘单位和审批部门确认。

Q5: SHP数据能替代测绘成果吗？

A: 不能。SHP服务定位为空间矢量数据整理与核对，不替代具有资质的测绘单位出具法定测绘成果。法定测绘成果需由持证测绘单位按测绘规范出具。

Q6: SHP文件制作需要多长时间？

A: 根据项目复杂度不同，一般需要2-5个工作日。建议在设计定稿后尽早启动SHP整理工作。
